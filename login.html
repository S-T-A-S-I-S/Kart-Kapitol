If you get blocked

Firebase login won’t work from file://. It must be served over HTTPS (GitHub Pages is perfect).

  • Admin-only event uploads
  • Live updates to Events page (Firestore)
  • Featured events for the “big wall”

Access is controlled by roles in Firestore. If you can’t access Admin, you need your UID added to roles as admin.
